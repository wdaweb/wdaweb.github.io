HTML

HTML 是基礎網頁技術，網頁瀏覽器可以讀取 HTML 檔案，成為視覺化網頁。

CSS

CSS 網頁樣式表，可彈性自定變化網頁元素顯示樣貌，節省工作時間。

JavaScript

JavaScript 是最流行的腳本語言，能在靜態頁面上添加一些動態效果。

jQuery

以 JS 為基礎的函式庫，號稱能讓開發者能寫的少，做的多！

Boostrap

熱門的前端框架工具，用於設計響應式網站、排版、格式和介面。

Photoshop

學習影像處理的主流軟體，用於規劃網頁版面或編修圖片應用。

Illustrator

熟悉貝茲曲線運用，不失真的向量圖形格式，廣泛的應用在網頁等設計。

Git

Git 為分散式版本控制系統，可以把檔案的狀態作為更新歷史記錄保存起來。

Node.js

Node.js 是能夠在伺服器端運行 JavaScript 的跨平台執行環境。

Vue.js

Vue.js 是一個用於建立使用者介面和單頁應用的 JavaScript 框架。

MongoDB

MongoDB 是文件導向的資料庫系統，為目前最靈活易用的 NoSQL 之一。

PHP

PHP 為可快速開發的腳本語言。在網站資料處理應用上被廣泛運用。

MySQL

MySQL 是一個小型關聯式資料庫管理系統，被廣泛地應用在中小型網站中。

Laravel

Laravel 是近年來相當熱門的PHP框架，用於製作網頁的 MVC 架構。
